# RapidForge

**A lightning-fast, zero-setup HTML, CSS, and JavaScript prototyping environment.**

RapidForge is a single-file web application that gives you a complete, professional IDE experience directly in your browser. It is perfect for rapid UI prototyping, debugging components, or learning frontend development without the hassle of setting up local servers or build tools.

## Features

* **Live Preview Engine:** Instantly renders your code as you type, safely sandboxed in an iframe.
* **Professional Editor:** Powered by CodeMirror with syntax highlighting, auto-close tags/brackets, and code folding.
* **Light & Dark Mode:** Native theme toggling that remembers your preference.
* **Responsive Testing:** Instantly switch between Desktop and Mobile preview environments.
* **Built-in Developer Console:** Intercepts and formats `console.log`, `console.warn`, and `console.error` directly in the UI.
* **One-Click CDN Injection:** Instantly add Tailwind CSS, Bootstrap 5, or jQuery to your environment.
* **Layout Debugger:** "Ghost Mode" outlines all elements on the page to help you fix CSS layout issues.
* **Export Ready:** Download your active environment as a clean `index.html` file with a single click.

## Quick Start

Because RapidForge is built entirely in vanilla web technologies, there are zero dependencies to install. 

1. Clone or download this repository.
2. Double-click `index.html` to open it in your browser.
3. Start coding!

## Deployment

You can host RapidForge for free in seconds. Since it is a static, single-page application, it works perfectly on edge networks and static hosts.

### Via GitHub Pages
1. Push this code to a new GitHub repository.
2. Go to **Settings** > **Pages**.
3. Under "Build and deployment", set the source to deploy from the `main` branch.
4. Your editor will be live at `https://[your-username].github.io/[repo-name]`.

### Via Netlify Drop
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the folder containing your `index.html` file.
3. Your live environment is instantly generated.
